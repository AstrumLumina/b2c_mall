#### 版本说明
* 软件版本周期介绍
  软件版本周期:

  * α、β、λ 常用来表示软件测试过程中的三个阶段。

  - α(alpha) 是第一阶段，一般只供内部测试使用；
  - β(Beta) 是第二个阶段，已经消除了软件中大部分的不完善之处，但仍有可能还存在缺陷和漏洞，一般只提供给特定的用户群来测试使用；
  - λ(Release Candidate) 是第三个阶段，此时产品已经相当成熟，只需在个别地方再做进一步的优化处理即可上市发行。
